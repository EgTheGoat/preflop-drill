# Poker GTO Trainer

テキサスホールデムの **プリフロップレンジ練習** ができる Web アプリ。
ランダムに席（ポジション）とハンドが出題され、選んだアクションをレンジと比較して採点する。

## 練習モード

| モード | 内容 |
|---|---|
| **GTO 9m** | 9-max 100bb キャッシュの RFI（教科書的な自作サンプル） |
| **GTO 6m** | 6-max 100bb キャッシュの RFI（教科書的な自作サンプル） |
| **ヨコサワ（トーナメント）** | 9-max。世界のヨコサワのレンジ表をベースにオープン / vsレイズ（コール・3bet）を出題 |

## 開発

```bash
mise install      # Node 24 を用意
npm install
npm run dev        # 開発サーバ
npm test           # ロジックのユニットテスト
npm run build      # 本番ビルド
```

## レンジデータについて

`src/data/ranges/*.json` のレンジは**教科書的な自作サンプル**であり、市販ツールの数値を複製していない。
形式は `src/types/range.ts` を参照。JSON を追加するだけで練習シナリオを増やせる。
